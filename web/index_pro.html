وب‌محور

امنیت پیشرفته

Frontend: HTML5, CSS3, JavaScript ES6+

Database: SQLite3 with indexing
Performance: Multi-threaded, async operations
Compatibility: Windows 10+, Linux, macOS
File Limit: Unlimited (Pro), 100 (Trial)
