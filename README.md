## 📋 معرفی

Persian File Copier Pro یک نرم‌افزار نوآورانه و حرفه‌ای است که برای مدیریت و کپی فایل‌ها با استفاده از پیشرفته‌ترین تکنولوژی‌های وب طراحی شده است. این برنامه قابلیت‌های منحصر به فردی مانند:

- 🚀 **تشخیص خودکار دستگاه‌ها** (USB، SSD، HDD، MTP)
- ⚡ **بهینه‌سازی سرعت کپی** برای انواع دستگاه‌ها
- 🎨 **تم‌های زیبا** با الهام از طبیعت ایران
- 📱 **رابط کاربری مدرن** و responsive
- 🔍 **جستجوی پیشرفته** و real-time
- 📊 **مدیریت تسک‌های حرفه‌ای**

## ✨ ویژگی‌های کلیدی

### 🔥 تکنولوژی‌های مدرن
- **FastAPI** برای backend پرسرعت
- **WebSocket** برای بروزرسانی real-time
- **SQLite** با ایندکس‌گذاری پیشرفته
- **jQuery** و **SweetAlert2** برای UI زیبا
- **Drag & Drop** و **Right-Click Context Menu**

### 🎨 تم‌های منحصر به فرد
- ☀️ **روشن** - تم کلاسیک و تمیز
- 🌙 **تاریک** - برای استفاده راحت شبانه
- 🏔️ **کوه البرز** - الهام از قله‌های مهیب ایران
- 🌊 **دریای خزر** - آرامش آبی خزر
- 🏜️ **کویر ایران** - گرمای طلایی کویرها
- 🏖️ **خلیج فارس** - سبزی زمردین خلیج

### ⚡ عملکرد بهینه شده
- **بافر دینامیک** بر اساس نوع دستگاه
- **کپی همزمان** تا 10 فایل
- **نمایش پیشرفت real-time** با سرعت و ETA
- **ایندکس‌گذاری سریع** فایل‌ها (کمتر از 20 ثانیه)

### 🔧 تنظیمات پیشرفته
- تأیید کپی با checksum
- حفظ تاریخ و مجوزهای فایل‌ها
- رد شدن از فایل‌های موجود
- گزارش‌گیری کامل

## 🔧 نصب و راه‌اندازی

### پیش‌نیازها
- **Python 3.8+** (توصیه: Python 3.11)
- **مرورگر مدرن** (Chrome, Firefox, Edge, Safari)
- **اتصال اینترنت** (برای CDN های مورد نیاز)

### راه‌اندازی سریع

1. **دانلود فایل‌ها:**
```bash

# - start.py
# - main.py  
# - config.py
# - index.html
```

2. **اجرای برنامه:**

3. **باز کردن مرورگر:**

   - یا دستی آدرس `http://localhost:8547` را باز کنید

### نصب دستی وابستگی‌ها (اختیاری)

## 🎯 نحوه استفاده

### 1. انتخاب درایو
- در پنل **درایوها** روی درایو مورد نظر کلیک کنید
- برنامه به‌طور خودکار فایل‌ها را ایندکس می‌کند

### 2. جستجو و فیلتر
- از **جستجوی سریع** استفاده کنید
- با **فیلترهای نوع فایل** نتایج را محدود کنید
- انتخاب **درایو خاص** برای جستجوی دقیق‌تر

### 3. انتخاب فایل‌ها
- **کلیک** روی checkbox برای انتخاب
- **Ctrl+کلیک** برای انتخاب چندگانه
- دکمه **☑️ انتخاب همه** برای انتخاب کلیه فایل‌ها

### 4. کپی فایل‌ها
- **Drag & Drop:** فایل‌ها را به درایو مقصد بکشید
- **Right-Click:** منوی کنتکست برای گزینه‌های بیشتر
- **Context Menu:** انتخاب "📋 کپی به مقصد"

### 5. مدیریت تسک‌ها
- پنل **تسک‌ها** نمایش پیشرفت real-time
- امکان **⏸️ توقف**، **▶️ ادامه**، **❌ لغو** تسک‌ها
- نمایش **سرعت** و **زمان باقیمانده**

## 🔌 API مستندات

برنامه یک API کامل ارائه می‌دهد:

- **GET** `/api/health` - بررسی سلامت سیستم
- **GET** `/api/drives` - لیست درایوها
- **POST** `/api/search` - جستجوی فایل‌ها
- **POST** `/api/copy` - شروع کپی
- **GET** `/api/tasks` - لیست تسک‌ها
- **GET** `/api/config` - تنظیمات برنامه
- **WebSocket** `/ws` - ارتباط real-time

مستندات کامل: `http://localhost:8547/docs`
